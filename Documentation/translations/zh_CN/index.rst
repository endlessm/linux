中文翻译
========

这些手册包含有关如何开发内核的整体信息。内核社区非常庞大，一年下来有数千名开发
人员做出贡献。 与任何大型社区一样，知道如何完成任务将使得更改合并的过程变得更
加容易。

翻译计划:
内核中文文档欢迎任何翻译投稿，特别是关于内核用户和管理员指南部分。
